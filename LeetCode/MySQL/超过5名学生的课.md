#### [超过5名学生的课](https://leetcode-cn.com/problems/classes-more-than-5-students/)

### 解题思路

`GROUP BY`  `HAVING`关键字

子查询

### AC代码

HAVING 条件查询

```mysql
SELECT 
    class
FROM 
    courses
GROUP BY 
    class
HAVING 
    COUNT(DISTINCT student)>=5
```

子查询

```mysql 
SELECT class
FROM 
    (SELECT 
        class,COUNT(DISTINCT student) as num
    FROM
        courses
    GROUP BY class) AS temp
WHERE num>=5
```
