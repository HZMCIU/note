### [只出现一次的数字](https://leetcode-cn.com/problems/single-number/)

### 解题思路

$$
a \oplus b \oplus a =b
$$

### AC代码

```cpp
class Solution
{
public:
    int singleNumber(vector<int>& nums)
    {
        int res=0;
        for(int d:nums)
        {
            res^=d;
        }
        return res;
    }
};
```
