#### [除自身以外数组的乘积](https://leetcode-cn.com/problems/product-of-array-except-self/)

### 思路

不能使用除法来计算

**前后缀**  `letf[i]` 表示$[0,i)$的乘积，`right[i]`表示$(i,n]$的成绩 `output[i]=right[i]*left[i]`

**空间优化** 使用`output[i]`来计算左前缀，右后缀再遍历过程中生成。

### AC代码

**前后缀**

```cpp
class Solution
{
public:
    vector<int> productExceptSelf(vector<int>& nums)
    {
        int n = nums.size();
        vector<int> left(n + 1);
        vector<int> right(n + 2);
        vector<int> output(n);

        right[n + 1] = 1;
        left[0] = 1;
        for (int i = 1; i <= n; i++)
            left[i] = left[i - 1] * nums[i - 1];

        for (int i = n; i >= 1; i--)
            right[i] = nums[i - 1] * right[i + 1];

        for (int i = 1; i <= n; ++i)
            output[i - 1] = left[i - 1] * right[i + 1];
        return output;
    }
};
```



**空间优化**

```cpp
class Solution
{
public:
    vector<int> productExceptSelf(vector<int>& nums)
    {
        int n = nums.size();
        vector<int> output(n);
        output[0] = 1;
        int r = 1;
        for (int i = 1; i < n; i++)
            output[i] = output[i - 1] * nums[i - 1];
        for (int i = n - 1; i >= 0; i--)
        {
            output[i] = output[i] * r;
            r *= nums[i];
        }
        return output;
    }
};

```
