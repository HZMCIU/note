### [剑指 Offer 19. 正则表达式匹配](https://leetcode-cn.com/problems/zheng-ze-biao-da-shi-pi-pei-lcof/)

请实现一个函数用来匹配包含`'. '`和`'*'`的正则表达式。模式中的字符`'.'`表示任意一个字符，而`'*'`表示它前面的字符可以出现任意次（含0次）。在本题中，匹配是指字符串的所有字符匹配整个模式。例如，字符串`"aaa"`与模式`"a.a"`和`"ab*ac*a"`匹配，但与`"aa.a"`和`"ab*a"`均不匹配。

### 思路

模式串`p`中包含三种可能的字符，`*`,`.`,正常字符。假设有状态`dp[i][j]`，代表`p[0..j]`和`s[0..i]`是否能够匹配。
设`s[i]`和`p[j]`为当前匹配过程中，源串和模式串中的最后一个字符，`p[j]`只有`*`,`.`,正常字符三种可能，分类讨论

+ `p[j]`为正常字符，如果`s[i]==p[j]`，那么`s[0..i]`和`p[0..j]`的匹配转化为**子问题** `s[0..i-1]`和`p[0..j-1]`的匹配，如果`s[0..i-1]`和`p[0..j-1]`匹配，那么`s[0..i]`和`p[0..j]`匹配。
+ `p[j]`为`.`，`.` 能够和任意字符匹配，`s[0..i]`和`p[0..j]`的匹配转化子问题`s[0..i-1]`和`p[0..j-1]`的匹配
+ `p[j]`为`*`，`p[j-1,j]='c*'`，`c*`是一个整体，`p[j-1]='c'`可以重复0次，或者多次
  + 重复0次，直接丢弃模式串的后两位，转化为子问题`s[0..i]`和`p[0..j-2]`的匹配
  + 重复多次，如果`s[i]==p[j-1]`，转化为子问题`s[0..i-1]`和`p[0..j]`的匹配

**初始条件**

1. 模式串和源串均为空，则源串和模式串是匹配的，即`dp[0][0]=true`
2. 模式串为空，源串非空，则源串和模式串均不匹配，即`dp[i][0]=false,i>0`



**状态转移方程**
$$
dp[0][0]=true \ \ ,\ \ dp[i][0]=false 
$$
$$
dp[i][j]=
\begin{cases}
	dp[i-1][j-1]\ \ ,\ \ s[i]==p[j] \ \  或 \ \  p[j]=='.'\\
	dp[i][j-2]\ \ ,\ \ p[j]=='*' \ \  且  \ \ 模式串重复0次  \\ 
	dp[i-1][j]\ \ ,\ \ p[j]=='*' \ \ 且  \ \ 模式串重复1..n次
\end{cases}
$$
### AC代码

```java
class Solution {
    public boolean isMatch(String s, String p) {
        int n = s.length();
        int m = p.length();
        // 数组开n+1方便处理
        boolean[][] dp = new boolean[n + 1][m + 1];
        for (int i = 0; i <= n; i++) {
            for (int j = 0; j <= m; j++) {
                if (j == 0) {
                    dp[i][j] = i == 0;
                } else {
                    if (p.charAt(j - 1) != '*') {
                        if (i >= 1 && (s.charAt(i - 1) == p.charAt(j - 1) || p.charAt(j - 1) == '.')) {
                            dp[i][j] = dp[i - 1][j - 1];
                        }
                    } else {
                        if (j >= 2) {
                            dp[i][j] |= dp[i][j - 2];
                        }
                        if (i >= 1 && j >= 2 && (s.charAt(i - 1) == p.charAt(j - 2) || p.charAt(j - 2) == '.')) {
                            dp[i][j] |= dp[i - 1][j];
                        }
                    }
                }
            }
        }
        return dp[n][m];
    }
}
```
