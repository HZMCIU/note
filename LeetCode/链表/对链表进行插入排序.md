### [对链表进行插入排序](https://leetcode-cn.com/problems/insertion-sort-list/)

### 题意

对链表进行插入排序。

### 思路

使用`lastSorted`指针维护有序序列的末尾结点。当`lastSorted`下一个结点非空，且小于有序列表的最后一个结点时，遍历有序列表进行插入。
+	`lastSorted.next`用来指示这一轮**待插入的结点**。如果单独使用`curr`指示待插入结点，在选择排序过程中，结点的后继会发生改变，那么**下一轮排序将无法找到待排序的结点**

使用`dummy`头节点，方便链表操作。

### AC代码

```java
class Solution {
    public ListNode insertionSortList(ListNode head) {
        if (head == null || head.next == null)
            return head;
        ListNode lastSorted = head;
        ListNode dummy = new ListNode();
        dummy.next = head;
        while (lastSorted.next != null) {
            // 进行插入排序
            if (lastSorted.next.val < lastSorted.val) {
                // 删除末尾的结点
                ListNode cur = lastSorted.next;
                lastSorted.next = lastSorted.next.next;

                // 从dummy开始遍历
                ListNode p1 = dummy;
                while (p1.next != null && p1.next.val <= cur.val) {
                    p1 = p1.next;
                }
                cur.next = p1.next;
                p1.next = cur;
            } else {
                lastSorted = lastSorted.next;
            }
        }
        return dummy.next;
    }
}
```
