#### [链表的中间结点](https://leetcode-cn.com/problems/middle-of-the-linked-list/)

### 解题思路

快慢指针

### AC代码

```java
class Solution {
    public ListNode middleNode(ListNode head) {
        ListNode fast=head,slow=head;
        while(fast.next!=null){
            if(fast.next!=null)
                fast=fast.next;
            if(fast.next!=null)
                fast=fast.next;
            if(slow.next!=null)
                slow=slow.next;
        }
        return slow;
    }
}
```
