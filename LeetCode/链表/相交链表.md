### [相交链表](https://leetcode-cn.com/problems/intersection-of-two-linked-lists/)

### 题意

编写一个程序，找到两个单链表相交的起始节点。

### 思路

+ 创建两个指针`pA`和`pB`，分别初始化为链表A和链表B的头节点。
+ 当`pA`遍历到链表A的尾部时，将`pA`重定位到链表B的头节点。`pB`同理。
+ 当`pA`和`pB`相遇时，相遇的结点便是相交结点

可以考虑以下两个链表: `A={1,3,5,7,9,11}` 和 `B={2,4,9,11}`，相交于结点 `9`。 由于 `B.length (=4) < A.length (=6)`，`pB` 比 `pA` 少经过2个结点，会先到达尾部。将`pB`重定向到 A 的头结点，`pA`重定向到 B 的头结点后，`pB`要比 `pA`多走 2 个结点。因此，它们会同时到达交点。

如果两个链表不相交，在`pA`和`pB`第一次重定位后，会同时到达链表末尾，也就是`null`，只要返回其中一个指针即可

### AC代码

```java
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        ListNode pA = headA, pB = headB;
        while (pA != pB) {
            pA = (pA == null ? headB : pA.next);
            pB = (pB == null ? headA : pB.next);
        }
        return pA;
    }
}
```
